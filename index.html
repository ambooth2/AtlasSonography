About Celestia

Celestia is a biomedical engineering initiative focused on developing accessible, intelligent diagnostic tools for real-world and remote environments.

Project Overview

Our project focuses on developing a portable ultrasound system enhanced with AI-driven diagnostics to assist users in medical imaging and interpretation.

Key Features

  • Portable ultrasound integration
  • Real-time AI assistance
  • Low-power operation
  • Offline functionality

Applications

Designed for space missions, remote healthcare environments, and emergency medical scenarios.
